# Arduino-Car
This project is an Arduino-based obstacle avoidance robot that uses four ultrasonic sensors to detect obstacles in multiple directions and navigate autonomously. The sensors continuously measure distances and send data to the Arduino UNO, which processes the inputs and controls two DC motors through an L293D motor driver to decide movement such as moving forward, turning, or stopping based on obstacle proximity. The system is designed for real-time decision-making and efficient navigation, making it suitable for robotics projects, competitions, and basic autonomous vehicle applications, with the complete circuit connections provided in the project files
